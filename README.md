# AI Portfolio Assistant

An intelligent portfolio assistant built with Django and OpenRouter API.

## Features
- AI-powered recruiter interaction
- Memory-based conversation
- Professional response control
- Custom frontend chat UI

## Tech Stack
- Django
- Python
- OpenRouter API
- HTML/CSS/JavaScript

## Setup

```bash
git clone https://github.com/yourusername/ai-portfolio-assistant.git
cd ai-portfolio-assistant
pip install -r requirements.txt
python manage.py runserver
